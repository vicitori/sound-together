# Sound Together
Простой веб-сайт где друзья могут вместе создавать один плейлист. 
1. каждый добавляет свои треки
2. голосует за лучшие
3. получается идеальный плейлист для встречи вместе

# Пользователи:
**Владелец плейлиста** - создаёт плейлист и делится ссылкой
**Друзья** - переходят по ссылке и добавляют свою музыку
